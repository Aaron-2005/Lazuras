How to use

1. Pick tiles or objects
2. Pick a tool
3. Draw on the grid
4. Copy the export text
5. Paste it into ChatGPT
6. I convert it into levels.py

Conventions

S = spawn
X = exit
T = gate
V = lever
P = plate
M = moving platform

Use one object per cell.
Use repeated T cells if you want to sketch a tall gate.
